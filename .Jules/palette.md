## 2024-05-18 - [Calculator Input Accessibility]
**Learning:** [Calculator displays should use readonly and aria-label attributes. readonly prevents the mobile keyboard from popping up on mobile devices when tapping the display input, providing a smoother experience. aria-label makes it clear for screen readers that the input is a calculator display.]
**Action:** [When implementing custom calculator or interactive displays, ensure inputs used only for output have readonly to prevent mobile keyboard popups and aria-label for accessibility.]
